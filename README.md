# Vibely 📱

A modern, responsive frontend for a social media application. Vibely allows users to navigate between an authentication screen, a dynamic feed of posts, and a personal user profile. Built with a strict mobile-first workflow to ensure a seamless experience across all devices.

## Live Demo

[Click here to view the live site!](https://aw3stm.github.io/css-frameworks/)

## 🛠️ Built With

This project was developed using modern frontend technologies:

- **HTML5** & **Vanilla JavaScript**
- **Tailwind CSS (v4)** - For rapid, utility-first styling.
- **Vite** - As the lightning-fast frontend tooling and build server.
- **GitHub Actions** - For automated CI/CD deployment to GitHub Pages.

## Features

- **Authentication UI:** Toggle seamlessly between Sign In and Register forms.
- **Social Feed:** A responsive timeline of posts featuring images, user avatars, and captions.
- **User Profile:** A dedicated profile page displaying user stats (followers/following) and a grid of their uploaded content.
- **Mobile-First Design:** Fully optimized for mobile screens with a bottom navigation bar, scaling gracefully up to desktop layouts with sidebars.
- **Modals:** Interactive UI elements for creating new posts and searching the platform.

## Getting Started

To run this project locally on your computer, follow these steps:

### Prerequisites

Make sure you have [Node.js](https://nodejs.org/) installed on your machine.

### Installation

1. Clone the repository:

   ```bash
   git clone [https://github.com/aw3stm/css-frameworks.git](https://github.com/aw3stm/css-frameworks.git)
   ```

   2. Navigate to the directory:
      cd css-frameworks

   3. Install dependencies:
      npm install

   4. Start local dev server:
      npm run dev

   5. Open your browser and visit http://localhost:5173/ (or the port specified in your terminal).

### This project is automatically deployed to GitHub Pages upon pushing to the main branch, utilizing a custom GitHub Actions workflow to build the Vite project and handle relative asset routing.
